API Documentation

Authentication & Authorization

All endpoints except /register and /login require JWT authentication in the Authorization header

Data Transfer Objects (DTOs)

RegisterRequest

FieldTypeDescription
usernameStringUnique username
passwordStringUser password
roleStringUser role (USER/ADMIN)
firstNameStringFirst name
lastNameStringLast name
sexStringGender
ageIntegerUser age
emailStringEmail address

LoginRequest

FieldTypeDescription
usernameStringRegistered username
passwordStringAccount password

API Endpoints

Authentication Endpoints

POST

/api/auth/register

Register a new user

Request Example:

{
    "username": "john_doe",
    "password": "securePass123",
    "role": "USER",
    "firstName": "John",
    "lastName": "Doe",
    "sex": "Male",
    "age": 30,
    "email": "john@example.com"
}
POST

/api/auth/login

User login

Response: JWT token

POST

/api/auth/logout

Logout user

User Management Endpoints

GET

/api/users

Get all users (Admin only)

Response Example:

[
    {
        "username": "john_doe",
        "role": "USER",
        "profile": {
            "firstName": "John",
            "lastName": "Doe",
            "sex": "Male",
            "age": 30,
            "email": "john@example.com"
        }
    }
]
POST

/api/users/change-password

Change user password

Request Example:

{
    "oldPassword": "oldPass123",
    "newPassword": "newSecurePass456"
}
PUT

/api/users/update-profile

Update user profile

Request Example:

{
    "firstName": "John",
    "lastName": "Smith",
    "email": "john.smith@example.com"
}
PUT

/api/users/change-username

Change username

Request Example:

{
    "newUsername": "john_smith"
}
DELETE

/api/users/{userId}

Delete user (Admin only)

Response:

{
    "message": "User account deleted successfully by admin"
}

Response Status Codes

CodeDescription
200OK - Successful operation
400Bad Request - Invalid input
401Unauthorized - Missing/invalid token
403Forbidden - Insufficient privileges
404Not Found - Resource not found
500Internal Server Error
